Dataset Class#

class servicex_client.dataset.Dataset(dataset_identifier: DataSetIdentifier | FileListDataset | None = None, title: str = 'ServiceX Client', codegen: str | None = None, sx_adapter: ServiceXAdapter | None = None, config: Configuration | None = None, query_cache: QueryCache | None = None, servicex_polling_interval: int = 10, minio_polling_interval: int = 5, result_format: ResultFormat | None = None)[source]#

This is the main class for constructing transform requests and receiving the results in a format of your choice. It can be run synchronously or asynchronously.

Parameters:
